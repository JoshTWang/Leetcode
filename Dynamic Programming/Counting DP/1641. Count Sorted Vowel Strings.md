#### DP problem

```java
class Solution {
    public int countVowelStrings(int n) {
        int[] curr = new int[5];
        Arrays.fill(curr, 1); // base case
        for (int i = 1; i < n; i += 1) {
            int[] next = new int[5];
            next[4] = curr[4]; // u - u
            next[3] = next[4] + curr[3]; // o - ou
            next[2] = next[3] + curr[2]; // i - iou
            next[1] = next[2] + curr[1]; // e - eiou
            next[0] = next[1] + curr[0]; // a - aeiou
            curr = next;
        }
        int res = 0;
        for (int i = 0; i < 5; i += 1) {
            res += curr[i];
        }
        return res;
    }
}
```

class Solution {
    public int countVowelStrings(int n) {
        return (n + 4) * (n + 3) * (n + 2) * (n + 1) / 24;
    }
}
