 [Digital Root](https://en.wikipedia.org/wiki/Digital_root)



#### Recursion

```java
class Solution {
    public int addDigits(int num) {
        if (num / 10 == 0) {
            return num;
        }
        int sumDigits = 0;
        while (num != 0) {
            sumDigits += num % 10;
            num /= 10;
        }
        return addDigits(sumDigits);
    }
}
```

class Solution {
    public int addDigits(int num) {
        if (num == 0) return 0;
        if (num % 9 == 0) return 9;
        return num % 9;
    }
}
